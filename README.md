# Tarobest

Tarobest 是一个的脚手架工具，它可以帮助你快速搭建一个 Taro 项目。

## 用法

```bash
npm install -g tarobest
tb init
```

## 本地开发

```bash
git clone https://github.com/Tarobest/Tarobest.git
cd Tarobest
pnpm i
pnpm run dev
# 新建终端
npm link
tb init
```
